# linrecopy
A Linux Utility ment to copy files on to a USB from within Linux recovery.

This can also run under MacOS, BSD and other UNIX based Operating Systems. This is a port of winrecopy.

# For Windows Users visit <a href="https://github.com/Adock90/winrecopy">here</a>

# Build 

Step 1:

download the git clone

<code>git clone https://github.com/Adock90/linrecopy/</a>

Step 2:

Run the ./build.sh

Step 3:

Run with <code>./linrecopy ~/ /media/user/USB #replace args for real paths</code>
